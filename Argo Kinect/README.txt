PRE-REQUISITES:
Kinect for Windows SDK 2.0 https://www.microsoft.com/en-us/download/details.aspx?id=44561 -used to communicate with the Kinect camera 
 
Point Cloud Library (PCL) http://pointclouds.org/ -used to process the point clouds from the Kinect camera. -the PCL framework contains algorithms including filtering, feature estimation, surface reconstruction, registration, model fitting and segmentation.  Follow the setup for Visual Studio 2015 here: http://ninankara.net/2016/02/24/capture-point-cloud-with-pcl-1-8-and-kinect-v2-1setup/ 
 
MySQL Server, MySQL Connector/C++  https://dev.mysql.com/doc/connector-cpp/en/ -used for communicating with MySQL servers and gain access to the database. 

INSTRUCTIONS:
-Install program by running "Argo Kinect.msi"

During Run Time:
-Run "Argo Kinect.exe"
-To view the results, go to the folder where it is installed and in the "results" folder. There are also sample results there. 
-You may also edit parameters there by editing "config.txt" but be warned of the effects on the program.

For more questions, consult the User Manual & Technical Manual
